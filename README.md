# e-plantShopping

This repository contains the `e-plantShopping` project, a front-end React application built with Vite.

## Project Overview

`e-plantShopping` is a shopping app focused on building a plant store UI. It demonstrates React fundamentals including component structure, state management, props, event handling, and styling for an interactive product browsing experience.

## Features

- Product listing and selection
- React component-based UI
- User interaction with shopping-like interface
- Built with Vite for fast development and bundling

## Repository

This repository is named `e-plantShopping` and includes the source files, configuration, and assets used for the project.
